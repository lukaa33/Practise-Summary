HTML & CSS Basics Summary

  1. HTML Summary

  2. HTML uses "elements" to describe (annotate) content

  3. HTML elements typically have an opening tag, content and then a closing tag

  4. You can also have void (empty) elements like images

  5. You can also configure elements with attributes

  6. Theres a long list of available elements but you will gain experience over time, no worries

  7. Learn more about all avilable HTML elements on the MDN HTML element reference.

  1. CSS Summary

  2. CSS (Cascading Style Sheets)is used for styling your page content.

  3. Styles are assigned via property-value pairs.

  4. You can assign styles via the "style" attribute

  5. To avoid code duplication, you typically use global styles(e.g. via "style" element) instead

  6. Alternatively, you can work with external stylesheet files which you "link" to

  7. When working with CSS, concepts like "inheritance", "specificity" and the "box-model" should be understood.

  8. Learn more about all available CSS properties and values on the MDN CSS property reference.
